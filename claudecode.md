Please initialize this project workspace with the following developer constraints:
- Ensure all generated UI pages follow the modern shadcn/ui and Tailwind dark-theme paradigm.
- Use zinc-based dark neutrals (zinc-950, zinc-900) instead of pure black (#000).
- Apply frosted-glass styling (backdrop-blur) and thin border accents to all container cards.
- Organize dashboard layouts into clean, asymmetrical Bento Grids.
- Always check if an existing shadcn primitive can be used before writing custom components.
